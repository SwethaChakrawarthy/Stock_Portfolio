# Stock Portfolio Manager 📈

A CLI-based stock portfolio manager built with Core Java.

## Features
- View market stocks
- Buy and sell stocks
- Track portfolio value
- View transaction history
- Case-insensitive stock search

## Technologies
- Java
- Maven
- Git and GitHub

## Data Structures Used
- HashMap — portfolio stock lookup
- Stack — transaction history (LIFO)

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run App.java
